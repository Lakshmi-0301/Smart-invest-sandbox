# Smart Invest Sandbox

A risk-free stock market simulation platform that allows users to start with a virtual balance of $100,000 to learn trading, practice investment strategies, and understand financial markets without real monetary risk.

---

## Features

### 1. Trading & Portfolio Management

* Virtual $100,000 starting balance
* Real-time stock trading with live market data
* Portfolio tracking with gains/losses and performance insights
* Complete transaction history
* Diversification and portfolio health analysis

### 2. Educational System

* Interactive tutorials with structured learning paths
* Beginner to advanced courses
* Quizzes and exercises to reinforce concepts
* Progress tracking with badges and certifications

### 3. AI-Powered Tools

* Market forecasting using LSTM neural networks
* AI chatbot assistant for stock and portfolio queries
* Portfolio risk assessment 

### 4. Market Intelligence

* Real-time stock prices and market movements
* Curated financial news feed
* Technical analysis tools (chart patterns, indicators)
* Fundamental analysis with company financials and valuation metrics

### 5. User Experience

* Responsive and intuitive dashboard
* Multi-level tutorial system
* Completely risk-free environment for practice

---

## Tech Stack

### Backend

* Java 23
* Spark Framework (REST API)
* SQLite database
* Maven for dependency management
* JSON & Gson for serialization

### Frontend

* React.js
* Vite
* CSS3 for responsive layouts and animations
* Chart.js for visualizations

### AI & Data

* LSTM models for price forecasting
* Real-time financial APIs
* WebSocket for live updates

## Getting Started

### Prerequisites

* Java 23 or higher
* Node.js 16+
* Maven 3.6+

---

## Installation

```bash
git clone https://github.com/Lakshmi-0301/Smart-invest-sandbox/
cd Smart-invest-sandbox
```

---

## Backend Setup

```bash
cd src/main/java/backend
mvn clean compile
mvn exec:java -Dexec.mainClass="backend.Main"
```

---

## Frontend Setup

```bash
cd src/frontend
npm install
npm run dev
```

---

## Access the Application

* Frontend: [http://localhost:3000](http://localhost:3000)
* Backend API: [http://localhost:8080](http://localhost:8080)
